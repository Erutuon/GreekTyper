Options:

  • Composed characters: If checked, use a single character for Greek letters with diacritics whenever possible: for instance, a) (1F00) instead of a + 0) (3B1 + 313). In technical terms, this means that Normalization Form Composition (NFC) is used, rather than Normalization Form Decomposition (NFD).
  • Deleting individual diacritics: If checked, backspace deletes one diacritic at a time. For example, a)/ → a) → a. If unchecked, backspace deletes a letter plus all its diacritics. (This is true whether the diacritics are combining or not.)
  • Putting diacritics in the correct order: if checked, Greek diacritics will be automatically sorted in the order macron or breve (0_, 0^); then rough breathing, smooth breathing, or diaeresis (0(, 0), 0+); then acute, grave, or circumflex accent (0/, 0\, 0=), then iota subscript (0|).
  • Validating diacritics: if checked, a letter can only have one of each type of diacritic. (Type refers to the groups of diacritics that are at the same position in the sort order described above.) This means that your input is ignored if the letter already has that type of diacritic. For instance, typing a// will produce a/, not a//, and typing a() will produce a(, not a().
  • Replacing diacritics: this modifies the behavior of validation. Instead of not being able to type another character, you can replace existing diacritics of the same type. For instance, typing a smooth breathing 0) ()) after an alpha with rough breathing a( (a() will produce alpha with smooth breathing a), because the smooth breathing and rough breathing are of the same type.
  • Allowing both macron and breve: if checked, a vowel can have both a macron and breve: a_^. When you uncheck this, any macron–breve sequences will be replaced with macrons: a_^ → a_.

Features:

  • Shift + backspace deletes a sequence of diacritics without deleting the base character. For example, a_)/| → a. (This works even if the composed characters setting is on.)

Notes:

  • Greek question mark ; (37E) is normalized to semicolon ; (3B) and Greek semicolon or colon · (387, ano teleia) to middle dot or interpunct · (B7), so the script uses the latter characters.
  • Several features of this script don't function yet in Microsoft Edge.
